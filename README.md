JParFor (Java Parallel For)
===========================
Multithreading implementation parallel "for" operator in Java.


Requirements
------------
 * Java 1.8 or newer.


Install
-------
Maven:

    <dependency>
        <groupId>org.jparfor</groupId>
        <artifactId>jparfor</artifactId>
        <version>1.1.29</version>
    </dependency>
